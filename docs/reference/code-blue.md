# CODE BLUE: Emergency Cold Weather Response

## Overview

CODE BLUE is Santa Fe's emergency protocol activated during dangerously cold weather to protect people experiencing homelessness from hypothermia, frostbite, and cold-related deaths. When activated, additional warming shelter capacity opens and coordinated outreach efforts intensify to bring people indoors.

CODE BLUE may also be activated based on:

- [ ] Wind chill bringing "feels like" temperature below threshold

- [ ] Extended duration of cold (multi-day events)
- [ ] National Weather Service advisories or warnings

**High-Risk Individuals:**

Review By-Name List for people with:

- Elderly (65+) or very young
- Medical conditions affected by cold
- Mobility limitations

- Recently seen in vulnerable locations

### Outreach Messaging

When contacting individuals:

1. **Inform** - "CODE BLUE is active. It's dangerously cold tonight."
2. **Offer** - "Warming shelters are open at [locations]. Can we give you a ride?"
3. **Respect** - If declined, offer supplies and inform of risks
4. **Document** - Log contact in By-Name List, note refusal if applicable
5. **Return** - Check back, especially overnight

## Deactivation Criteria

CODE BLUE is deactivated when:

- [ ] Temperature rises above [threshold] for [#] consecutive hours
- [ ] No precipitation and wind chill above threshold
- [ ] National Weather Service lifts cold weather advisories
- [ ] CODE BLUE Coordinator officially announces deactivation

### Deactivation Process

1. Coordinator announces deactivation via [communication channels]
2. Warming shelters return to normal operating hours within [# hours]
3. Outreach teams resume normal schedules
4. Final check on high-risk individuals
5. Supplies inventoried and restocked

## After-Action Review

After each CODE BLUE activation, conduct a review within [7 days]:

### Review Questions

1. **Activation**: Was the decision timely? Were providers notified effectively?
2. **Capacity**: Did we have enough warming shelter beds? Were there turnaways?
3. **Outreach**: Did we reach high-risk individuals? What areas were missed?
4. **Transportation**: Were there gaps in getting people to shelter?
5. **Supplies**: What did we run out of? What wasn't needed?
6. **Communication**: Did internal/external communication work?
7. **Outcomes**: Any cold-related injuries or deaths? Any close calls?
8. **Improvements**: What should we do differently next time?

### Documentation

- [ ] Number of people served at each warming shelter
- [ ] Number of outreach contacts made
- [ ] Number of transports provided
- [ ] Supplies distributed
- [ ] Any incidents or injuries
- [ ] Weather conditions during activation
