## Notes

**Consistent Messaging Across All Communications:**

1. **City operates, Many Paths supports**  
   This is not a startup taking over. It's infrastructure the City owns.

2. **Platform license model, like MyChart**  
   Replicable, sustainable, not a one-off custom build.

3. **We're filling gaps, not replacing systems**  
   UniteUS, HMIS, Coalition all continue their work. We track the missing population.

4. **Data drives better outcomes**  
   More visibility = better coordination = higher success rates.

5. **Human-centered, not algorithm-driven**  
   Technology supports people, doesn't replace them.
